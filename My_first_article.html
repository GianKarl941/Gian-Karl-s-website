Vim is a very extensible program

Vim it's not only used for editing code in a text file, you can used Vim to edit a document by learning Latex or troff/groff, or edit a spreadsheet file using Vim, and etc. Vim can also expand it's abilities by creating a .vimrc file (for Vim) or init.vim (for Neovim), You can put whatever you want in your config file and if you are done adding stuff in your config file, save it and relaunch Vim. It's that simple.

Also you can add plugins to your vimrc file or init.vim file. This is where you can turn Vim into an IDE or microsoft office like enviroment, There are plenty of plugins for Vim and all you have to do is to start searching them on google or on duckduckgo and click the link to the website and start finding some cool plugins.
