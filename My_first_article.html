Why i think vim should be the main text editor for everyone

Nowadays, people use whatever application that they see on the mobile devices or on their computers. Especially when it comes to editing documents, people would rather use the Microsoft office suite than any other text editor. It suprises me the fact that people are being forced to use proprietary software when there are plenty of alternatives that you can use to create a document. Especially when it come to taking simple notes on your computers, your are better off typing your notes in markdown in a text editor rather than any other application that does not use a markup language.

Is there's any reason to use plain text to edit documents? Yes there is a reason why. Mainly for some people, the contents of the document is more important than the visuals. But the rest still prefer a stylistic document over a good written document. It just weird for me that people say that they need "The Adobe suite" when there's a lot of free and open source software that you can use for free and at the same time "free as in freedom, not free as free beer". Vim is one of these free and open source software that you should try to learn

Vim is a very extensible program

Vim it's not only used for editing code in a text file, you can used Vim to edit a document by learning Latex or troff/groff, or edit a spreadsheet file using Vim, and etc. Vim can also expand it's abilities by creating a .vimrc file (for Vim) or init.vim (for Neovim), You can put whatever you want in your config file and if you are done adding stuff in your config file, save it and relaunch Vim. It's that simple.

Also you can add plugins to your vimrc file or init.vim file. This is where you can turn Vim into an IDE or microsoft office like enviroment, There are plenty of plugins for Vim and all you have to do is to start searching them on google or on duckduckgo and click the link to the website and start finding some cool plugins.
